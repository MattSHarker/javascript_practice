# JavaScript Practice
A series of program to help teach myself javascript. 

The programs are created by following along with the book Eloquent Javascript, 3rd Edition by Marijn Haverbeke.


---
## Directory/File/Content listing
1. Types_and_operators
    1. types.js
        * Numbers
        * Strings
        * Booleans
        * Empty values
        * Automatic type conversion
    2. operators.js
        * Unary operators
        * Binary operators
        * Ternary operator
        * Boolean operators
        * Logical operators

2. program_structure
    1. expressionsAndStatements.js
        * Statements
        * Bindings
        * let keyword
        * var keyword
        * const keyword
    2. functions.js
        * Functions
    3. controlFlow.js
        * If-else
        * While and Do loops
        * For loops
        * break keyword
        * continue keyword
        * Switch case
